# Klartext
**This Spicetify extension simply removes gendering in the German language, should that annoy you too.**

<br/>

![Screenshot](https://github.com/Xeralin/Klartext/blob/main/Example.png)

## Installation
To install the extension, first download Klartext.js from [here](https://github.com/Xeralin/Klartext/blob/main/Klartext.js). Then you need to enter a few commands in the terminal.
First, open the Spicetify folder:
```bash
spicetify config-dir
```
Then place the Klartext.js file in the **Extension** folder.
Finally, you need to adjust the configuration:
```bash
spicetify config extensions Klartext.js
spicetify apply
```

<br/>

## Uninstallation
If you want to remove the extension, you can enter the following in the terminal:
```bash
spicetify config extensions Klartext.js-
```
